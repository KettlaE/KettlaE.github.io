<嗜血代码>游戏简评

Kettla · 2019-10-6 · 次阅读

​ 当初买这游戏是为了不错过预购奖励。~主要为了捏人（不是）~。这几天玩下来终于是通关了这游戏，总的来说流程不长，有惊喜也有遗憾吧。

缺点

想学却没学到到位的地图设计

在地图设计方面，很明显能看到魂系列的影子，充满了伪装成死人的敌人以及捷径门、电梯等设计，也同样完美还原了魂系游戏的死亡掉落，以及敌人的刷新机制（包括向下跳的深坑）。除此之外，《CODE VEIN》还加入了贴心的小地图，给地图的探索与物品收集带来了非常多的帮助。

但是不得不说游戏的地图衔接实在是太粗糙了，往往两片大地图之间是一段弯弯绕绕的羊肠小路，在途中无法看到任何变化。我明白这是给地图的加载提供时间，但是我认为这种枯燥的赶路甚至比不上直接提供一个传送点，简简单单的给个加载画面来的体验舒适。

打个比方，从雪山到荒漠那段路就特别奇怪，首先是一段很长的楼梯，然后就在一个漫长的通道里绕上三圈，这时候你突然发现前面亮了，出去一看，卧槽，瞬间就从冰天雪地转到荒漠迷城。我是完全不能接受的。可以说，基本上每一段地图都是割裂的，就如闯关游戏；既然如此，还不如直接篝火传送来得简单直白一些。如果开发者是想要营造一个完整的世界，那毫无疑问是失败的。

BOSS存在感不够

和传统黑魂不同的是，本作角色的升级并不能选择能力提升方向，而是全能力的提升。升级所花的雾粒随等级提升消耗量的提升也不是特别快，导致的结果是升级更容易，基本上打上几只怪就能够升上一级，这也使得玩家能力快速提升，一切都能靠刷解决问题，而想要不刷打败敌人的难度也大大提高了。这也就导致boss的难度对于不同的玩法可能完全不同。像我这种纯普攻大剑带队友的打法，可能能打到boss完全直不起身子，而如果想要单人过boss，可能会饱受折磨。而对我来说除去最终boss，几乎没有boss能给我留下什么深刻的印象，他们更像是精英怪，而不是boss。

优点

丰富的技能组合搭配

《CODE VEIN》引入了一种名为“血码”的职业系统，每个职业都会配有相应的技能，而这些技能通过熟练度的提升可以从专属技能转变为通用技能，大大提高了技能的搭配灵活性，外加上游戏内总和上百的技能，可以产生众多的玩法。

好用的NPC队友（工具人万岁）

本作的NPC除了在剧情中扮演重要角色外，还是我们通关的最大帮助。他们会在探索过程中进行语音互动，极大的降低了探索冗杂地图时跑路带来的枯燥感。

最重要的还是NPC能在玩家生命值归零时使用技能将生命值分给玩家，而且不限次数，只要技能冷却完成就能再次自动使用。

不错的游戏剧情

《CODE VEIN》的剧情并没有采用魂系列的叙事方法，故事简单但有着极为到位的角色塑造，包括每一段过场CG的演出都有极为合适的音乐搭配，人物动作和表情等细节也做得很不错，每个人物都有自己的独特个性，鲜明而令人印象深刻。

此外，玩家可以通过各种方式收集到名为血英的道具，可以用来解锁血码的同时看到血码主人的回忆，这对于丰富角色形象和完善游戏世界观起到了不小的作用，尽管在回忆世界的漫步略有拖沓，但也同样为对于不关心游戏剧情的人提供了跳过选项。可以说是十分贴心的了。

总结

加分：

  • 精美的人物建模和捏人系统
  • 较为优秀的剧情设计
  • 百听不厌的出色配乐
  • 独创性的丰富技能组合系统
  • 对玩家友好的NPC系统

减分：

  • 诡异的地图设计
  • 某些莫名奇妙的键盘键位
  • NPC礼物有BUG
  • 武器平衡性有待提升

二次元游戏中的上乘佳作，某些部分过于刻意模仿魂系列，但瑕不掩瑜，依旧有许多出彩之处。
